# SecureVault-C++-File-Encrypter-GUI

This is a GUI desktop application, build for WINDOWS. 
SecureVault is capable of encrypting multiple files at once, through a batch process. Currently it encrypts with AES-256, although plans to implement multiple technologies are in the works.
Users may also get hashes of these files. 

Users may decrypt the same files. All of this is done with a master password.
